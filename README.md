# Rahul's GenAI Agents (Daily Drops)

This is Rahul's personal repo for learning-by-building with GenAI agents.

From today onward, I'm publishing new agent work here regularly (aim: **one new agent file per day**). Each entry is intentionally small and focused so it stays easy to read, run, and iterate on.

## What's in here

- **Agent notebooks/scripts**: each agent lives in a single file (most often a Jupyter notebook).
- **Pinned dependencies**: Python packages are tracked in `requirements.txt`.

Current files:

- `simple_conversational_agent.ipynb` — a minimal conversational agent starter.

## Run locally (Python)

1. Create and activate a virtual environment.
2. Install deps:

   ```bash
   pip install -r requirements.txt
   ```

3. Start Jupyter and open a notebook:

   ```bash
   jupyter notebook
   ```

## Notes

- Some agents may require API keys (depending on the notebook). Set environment variables as needed.
- This repo is a learning log; code and structure can change as I upload daily.
